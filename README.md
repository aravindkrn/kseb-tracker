# ⚡ KSEB Tracker PWA

Track your KSEB LT-1 Domestic electricity bill — TOD meters, telescopic/non-telescopic tariffs, real-time projections.

## Quick Start
1. Open in **Chrome** on Android: `https://aravindkrn.github.io/kseb-tracker/`
2. Tap **⋮ → Add to Home Screen**

## Features
- 📟 Log T1/T2/T3 readings with edit/delete
- 💰 Real-time bill (EC + FC + Duty + FSC)
- 🔮 End-of-cycle projection
- 📊 4 animated charts (daily, bill trend, TOD pie, prediction vs actual)
- ⚙️ Profile/Settings (start date, readings, tariff mode)
- 🔔 Daily push notification reminder
- 📲 One-tap WhatsApp share
- 💡 Tips to reduce bill
- 🌐 Offline-capable via Service Worker
- 💾 All data stored locally — no server needed

## Important note on tariff mode
Do not choose tariff mode based only on the existence of T1/T2/T3 readings. Choose the exact mode as billed by KSEB.
